## 🚀 GETTING STARTED

### Installation
```bash
git clone https://github.com/skylerblue333/finishedforgoogle.git
cd finishedforgoogle
pnpm install
```

### Development
```bash
pnpm dev              # Start dev server
pnpm build            # Build for production
pnpm test             # Run tests
pnpm check            # TypeScript check
pnpm format           # Format code
```

### Database Setup
```bash
# Using built-in Manus database (recommended)
# Or local MySQL
pnpm db:push          # Generate migrations
```

### Deployment
```bash
# Deploy to Manus (recommended)
vercel deploy

# Or Docker
docker build -t skycoin4444 .
docker run -p 3000:3000 skycoin4444
```
